# 知乎自动发布 Agent

This skill was generated by Agent Builder. Use it to execute the workflow below in a consistent way.

## Goal

自动将准备好的文章发布到知乎专栏，并记录发布结果。

## Trigger

按计划定时触发（如每天 09:00）或手动触发。

## Inputs To Collect

- 文章标题
- 文章正文（Markdown 或富文本）
- 封面图（可选）
- 标签/话题（可选）
- 知乎登录状态（Cookie/已登录浏览器）

## Outputs To Produce

- 知乎文章发布链接
- 发布时间
- 发布状态（成功/失败）
- 失败原因与重试建议
- 发布日志

## Default Steps

1. 读取待发布文章内容
2. 校验必填字段（标题/正文）
3. 将 Markdown 转换为知乎编辑器可接受格式（如需）
4. 打开知乎创作中心并填充标题、正文、标签
5. 提交发布并等待结果
6. 抓取发布后的文章链接
7. 写入发布日志（本地文件/表格）
8. 发布失败时执行重试与告警

## Execution Shape

Use one agent session for this workflow unless the user explicitly asks to split the work further.

Do not create coordination overhead without a clear benefit.

## Approval Gates

- Ask for approval before: 首次发布前审批（检查标题/正文/标签）
- Ask for approval before: 检测到敏感词时强制审批
- Ask for approval before: 发布失败重试超过 1 次时审批
- Ask for approval before: 出现验证码或风控时人工接管

## Constraints

- 不绕过知乎安全机制，不进行违规自动化
- 首次登录或验证码场景需人工接管
- 发布前默认进行内容安全与敏感词检查
- 外部发布动作前可配置人工审批

## Success Criteria

在无需人工介入的情况下，按触发计划稳定发布；单次发布成功率 ≥ 95%，失败有明确日志与告警。
